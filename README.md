TermuCoderAI
Локальный AI-помощник для разработки и работы с языковыми моделями через Termux.
TermuCoderAI — проект для запуска и управления локальными LLM на Android-устройствах без использования облачных API.
На текущем этапе проект предоставляет CLI-инструменты для настройки окружения, управления конфигурацией и взаимодействия с локальным AI-сервером.
Возможности
Автоматическая настройка окружения
Управление конфигурацией проекта
Работа с локальными LLM-моделями
Подключение к llama-server
Работа через Termux на Android

Установка
Клонирование проекта
git clone https://github.com/vasilijpavlov693-arch/TermuCoderAI.git

cd TermuCoderAI
Установка зависимостей
pkg update && pkg upgrade

pkg install python git

pip install requests
Настройка
Для полной настройки проекта:
termucoder setup --full
Команда выполняет:
проверку Python;
установку зависимостей;
создание необходимых директорий;
проверку llama-server;
подготовку конфигурации.
Конфигурация
Настройки доступны через:
termucoder config
Основной файл:
settings.json
Использование
После запуска локального AI-сервера:
llama-server
TermuCoderAI подключается к серверу и отправляет запросы локальной модели.
