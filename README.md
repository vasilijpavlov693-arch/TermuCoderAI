# TermuCoderAI

Локальный AI-помощник для разработки и работы с языковыми моделями через
Termux.

TermuCoderAI предназначен для запуска локальных LLM на
Android-устройствах без использования облачных API.

Проект предоставляет CLI-инструменты для настройки окружения, управления
конфигурацией и взаимодействия с локальным AI-сервером.

------------------------------------------------------------------------

## Конфигурация

Настройки доступны через:

    termucoder config            # показать все настройки
    termucoder config show generation.temperature
    termucoder config set generation.temperature 0.4
    termucoder config set model.name my-model.gguf

Файл конфигурации:

    settings.json

Доступные секции: server, model, generation, prompts, user.

------------------------------------------------------------------------

## Использование

После запуска локального LLM-сервера:

    termucoder server start      # или вручную: llama-server -m <model>

TermuCoderAI подключается к серверу и отправляет запросы локальной
модели.

Основные команды:

    termucoder --version         # версия
    termucoder --help            # справка
    termucoder ask "объясни этот код"
    termucoder chat              # интерактивный чат (история сохраняется)
    termucoder config show
    termucoder doctor            # диагностика системы

Интерактивный чат поддерживает команды:

    /exit     выход
    /clear    очистить историю
