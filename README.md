# TermuCoderAI

Локальный AI-помощник для разработки и работы с языковыми моделями через
Termux.

TermuCoderAI предназначен для запуска локальных LLM на
Android-устройствах без использования облачных API.

Проект предоставляет CLI-инструменты для настройки окружения, управления
конфигурацией и взаимодействия с локальным AI-сервером.

------------------------------------------------------------------------

## Возможности

-   Автоматическая настройка окружения
-   Управление конфигурацией проекта
-   Работа с локальными LLM-моделями
-   Подключение к llama-server
-   Поддержка Termux / Android
-   CLI-интерфейс управления

------------------------------------------------------------------------

## Архитектура проекта

    TermuCoderAI

## Установка

### Клонирование

    git clone https://github.com/vasilijpavlov693-arch/TermuCoderAI.git
    cd TermuCoderAI

### Зависимости

    pkg update && pkg upgrade
    pkg install python git
    pip install requests

### Полная настройка

    termucoder setup --full

------------------------------------------------------------------------

## Конфигурация

Настройки доступны через:

    termucoder config

Файл конфигурации:

    settings.json

------------------------------------------------------------------------

## Использование

После запуска локального LLM-сервера:

    llama-server

TermuCoderAI подключается к серверу и отправляет запросы локальной
модели.
